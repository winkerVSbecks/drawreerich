drawreerich

A voxel drawing app where you sketch paths on a 2D grid and extrude them into 3D. It uses generative colour palettes. Tweak camera and artboard settings, and export your creations as JSON or PNG.

Created by

Varun Vachhar

Built with

  • heerich — 3D voxel scene rendering
  • ssam — sketch lifecycle
  • Tweakpane — UI controls
  • rampensau — palette generation
  • culori — colour handling

Source

github.com/winkerVSbecks/drawreerich
